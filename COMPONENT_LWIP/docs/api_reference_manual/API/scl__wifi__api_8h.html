scl_wifi_api.h File Reference

General Description

Prototypes of functions for controlling the Wi-Fi system

This file provides prototypes for end-user functions, which allow actions such as scanning for Wi-Fi networks, joining Wi-Fi networks, getting the MAC address, and so on.
